# DSA Practice

A collection of Data Structures and Algorithms problems and solutions.

## About

This repository contains my journey through various DSA problems, including:

- **Array & String Problems**
- **Linked List Implementations**
- **Tree & Graph Algorithms**
- **Dynamic Programming Solutions**
- **Sorting & Searching Algorithms**
- **Backtracking Problems**

## Structure

```
DSA-Practice/
├── Arrays/           # Array-based problems
├── Strings/          # String manipulation problems
├── LinkedList/       # Linked list implementations
├── Trees/            # Binary trees, BST, traversals
├── Graphs/           # Graph algorithms
├── DP/               # Dynamic programming solutions
├── Backtracking/     # Backtracking problems
├── Sorting/          # Sorting algorithms
├── Searching/        # Searching algorithms
└── DailyQue/         # Daily challenge problems
```

## Languages

- Python
- Java
- C++

## Daily Challenges

Solutions to daily coding challenges can be found in the `DailyQue/` folder, organized by date.

## Getting Started

Clone the repository:
```bash
git clone https://github.com/cdHarshita/DSA-Practice.git
cd DSA-Practice
```

Browse through the categorized folders to find solutions organized by topic.
